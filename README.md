# INSURANCE-RISK-CLAIMS-ANALYSIS

This Power BI dashboard offers an in-depth analysis of insurance policies, claims, and related risk factors. This report allows insurance companies to track claim frequency, claim severity, customer demographics, and risk distribution, thus improving profitability for the company.

This dashboard converts raw insurance data into useful information through interactive visuals and performance KPIs.

#Key Technologies Used

1. Power BI Desktop (.pbit template)

2. Power Query – Data cleaning, transformation, and preprocessing

3. DAX (Data Analysis Expressions) – KPI calculations and custom measures

4. Data Modeling – Relationship building and schema optimization

5. Interactive Visualizations – KPI cards, slicers, trend charts, bar and pie charts

6. Date Table Creation – Time intelligence analysis

#Data Source

The dashboard is built using a structured Insurance Risk & Claims dataset, which includes:

1. Policyholder demographics

2. Policy type and coverage details

3. Premium amounts

4. Claim amount and claim status

5. Risk categories

6. Policy duration

7. Claim frequency and severity indicators
